# Programming-Tutorial

**Programming tutorial using C++**

This programming tutorial aims to bring newcomers to programming and C++
a thorough introduction on how to write useful programs for the real world.

If your aim is to become a game programmer or game engine programmer; this turorial caters
to you. We'll cover basic syntax and problems to solve using the console window to start off with.
Later on in the tutorial, we'll cover how to request a window using the Win32 API and render objects onto the screen
using DirectX12.
